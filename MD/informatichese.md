# Lezione 1 – Fondamenti di informatica e pensiero computazionale

## Cos’è l’informatica
L’informatica è la scienza che studia il trattamento automatico dell’informazione:
- come i dati vengono rappresentati
- come vengono elaborati tramite algoritmi e programmi
- come vengono memorizzati, organizzati e trasmessi

Aree principali:
- Programmazione e algoritmi  
- Architettura dei calcolatori  
- Sistemi operativi  
- Basi di dati  
- Reti informatiche  
- Sicurezza informatica  

## Dato e informazione
- **Dato**: elemento grezzo (numero, parola, misura) senza contesto  
- **Informazione**: dato interpretato e reso significativo

## Bit e byte
- **Bit (b)**: unità minima di informazione (0 o 1)  
- **Byte (B)**: 8 bit  

Multipli:
- 1 KB ≈ 1 024 B  
- 1 MB ≈ 1 024 KB  
- 1 GB ≈ 1 024 MB  
- 1 TB ≈ 1 024 GB  

## Algoritmi
Un **algoritmo** è una sequenza finita di istruzioni non ambigue che, a partire da un input, produce un output.

Caratteristiche:
- Finitezza  
- Non ambiguità  
- Presenza di input e output  
- Ogni passo eseguibile in tempo finito  

## Pensiero computazionale e coding unplugged
Il **pensiero computazionale** è la capacità di:
1. Analizzare un problema (astrazione)  
2. Progettare una soluzione (algoritmi, automazione)  
3. Valutare il risultato (correttezza, efficienza)

**Coding unplugged**: attività senza computer per sviluppare logica e problem solving (schemi, giochi, carte, istruzioni passo-passo).


# Lezione 2 – Hardware, architettura del computer e memoria

## Cos’è un computer
Un computer è una macchina programmabile che:
- riceve dati (input)  
- li elabora secondo istruzioni (software)  
- restituisce risultati (output)

## Hardware e software
- **Hardware**: componenti fisici (CPU, RAM, disco, scheda madre, periferiche…)  
- **Software**: programmi e sistemi operativi che comandano l’hardware  

Esempi di software:
- Sistema operativo: Windows, macOS, Linux  
- Programmi applicativi: elaboratore testi, fogli di calcolo, browser, email  

## Architettura di Von Neumann
Struttura logica dei computer moderni:
- **CPU** (unità centrale di elaborazione)  
- **Memoria centrale**  
- **Dispositivi di input/output (I/O)**  
- **Bus** di comunicazione  
- **Unità di controllo**

## Componenti principali del PC
- Scheda madre  
- CPU  
- RAM  
- Unità di archiviazione (HDD/SSD)  
- Scheda video  
- Alimentatore  
- Case  

## HDD vs SSD
**HDD**:
- Dischi magnetici rotanti  
- Più economici a parità di spazio  
- Più lenti e più fragili

**SSD**:
- Memoria flash  
- Molto più veloci  
- Silenziosi, consumano meno  
- Più costosi a parità di GB

Uso tipico: SSD per sistema e programmi, HDD per archiviazione dati.

## RAM: tipi e formati
- **SRAM**: veloce e costosa, usata come cache  
- **DRAM**: meno costosa, usata come RAM principale  

Generazioni: DDR3, DDR4, DDR5.  
Formati:
- DIMM per desktop  
- SODIMM per notebook  


# Lezione 3 – Sistema operativo e gestione dei file

## Cos’è un sistema operativo
Il **sistema operativo (SO)** è il software di base che:
- gestisce l’hardware (CPU, RAM, periferiche)  
- organizza file e cartelle  
- permette l’esecuzione dei programmi  
- offre un’interfaccia all’utente (grafica o testuale)

Esempi: Windows, macOS, Linux.

## File e cartelle
- **File**: insieme di dati salvato con un nome e un’estensione (es. `.docx`, `.xlsx`, `.pdf`)  
- **Cartella**: contenitore di file e altre cartelle  

Operazioni di base:
- creare, rinominare, copiare, spostare, cancellare  
- organizzare documenti in modo logico (per materia, progetto, data…)

## Estensioni dei file
L’estensione indica il tipo di file:
- `.docx` → documento di testo  
- `.xlsx` → foglio di calcolo  
- `.pptx` → presentazione  
- `.pdf` → documento portabile  
- `.jpg`, `.png` → immagini  

## Competenze operative di base
- usare desktop, icone, menù, barra delle applicazioni  
- gestire pendrive / dischi esterni  
- usare correttamente salvataggio e “salva con nome”  
- fare backup periodici dei dati importanti  


# Lezione 4 – Elaborazione testi (Word o equivalenti)

## Cos’è un word processor
Un **elaboratore di testi** permette di:
- creare documenti  
- formattare testo e paragrafi  
- inserire immagini, tabelle, elenchi  
- impaginare e stampare / esportare in PDF

## Funzioni principali
- Formattazione carattere: grassetto, corsivo, sottolineato, colore  
- Formattazione paragrafo: allineamento, rientri, spaziatura, elenchi puntati/num.  
- Stili: Titolo 1, Titolo 2, Corpo del testo…

## Struttura di un documento
- Titolo  
- Sottotitoli / sezioni  
- Testo principale  
- Immagini / tabelle  
- Intestazione e piè di pagina (data, numero pagina, nome documento)

## Scorciatoie utili (Win)
- CTRL + C: copia  
- CTRL + V: incolla  
- CTRL + X: taglia  
- CTRL + Z: annulla  
- CTRL + S: salva  

## Esportazione in PDF
Dal menù “File” → “Esporta” o “Salva con nome” → selezionare formato `.pdf`.


# Lezione 5 – Fogli di calcolo (Excel o equivalenti)

## Cos’è un foglio di calcolo
Un foglio di calcolo permette di:
- organizzare dati in tabelle  
- eseguire calcoli automatici  
- creare grafici e riepiloghi  

## Struttura
- Celle (intersezione di riga e colonna)  
- Righe (1, 2, 3, …)  
- Colonne (A, B, C, …)  
- Fogli multipli nello stesso file

## Formule e operatori
Le formule iniziano con `=`.

Operatori:
- `+` addizione  
- `-` sottrazione  
- `*` moltiplicazione  
- `/` divisione  

Esempi:
- `=A1+B1`  
- `=(A1+A2)/2`

## Funzioni di base
- `=SOMMA(A1:A5)`  
- `=MEDIA(B1:B10)`  
- `=MIN(C1:C10)`  
- `=MASSIMO(C1:C10)`  
- `=CONTA(A1:A100)`  

Esempi leggermente più avanzati:
- `=ARROTONDA(3,14159; 2)` → 3,14  
- `=POTENZA(2; 3)` → 8  

## Tabelle e grafici
- Ordinare i dati per colonna  
- Applicare filtri  
- Creare grafici (colonne, linee, torta) a partire da una selezione di celle  


# Lezione 6 – Browser, motori di ricerca e posta elettronica

## Browser web
Un **browser** è un programma per navigare in Internet.

Funzioni principali:
- inserire URL nella barra degli indirizzi  
- aprire più schede  
- usare preferiti/segna-pagine  
- consultare cronologia  
- scaricare file

## Motori di ricerca
Un **motore di ricerca** permette di trovare informazioni usando parole chiave.

Esempi:
- Google  
- Bing  
- DuckDuckGo  
- Ecosia  

Concetti base:
- usare parole chiave significative  
- aggiungere parole per restringere la ricerca  
- tra virgolette per cercare una frase esatta  

## Navigazione in incognito
La modalità in incognito:
- non salva cronologia, cookie e moduli sul dispositivo  
- non rende invisibili al fornitore di connessione o ai siti web

## Posta elettronica
Con un account email è possibile:
- inviare e ricevere messaggi  
- allegare file  
- organizzare le email in cartelle / etichette  

Elementi di una email:
- Destinatario  
- Oggetto  
- Corpo del messaggio  
- Allegati  

Buone pratiche:
- oggetto chiaro e sintetico  
- stile educato e professionale  
- attenzione a mittenti sospetti e link strani  


# Lezione 7 – Reti, Internet e servizi online

## Cos’è una rete
Una rete informatica è un insieme di dispositivi collegati che possono:
- scambiarsi dati  
- condividere risorse (file, stampanti, connessione Internet)

## LAN e WAN
- **LAN (Local Area Network)**: rete locale (casa, ufficio, scuola)  
- **WAN (Wide Area Network)**: rete geografica estesa; collega più LAN

Internet è la più grande WAN esistente.

## Tipi di collegamento
- Cablato (Ethernet)  
- Wireless (Wi-Fi)  
- Reti mobili (4G, 5G, ecc.)

## Internet e servizi principali
Internet permette:
- navigazione web  
- email  
- servizi cloud  
- social network  
- chat e videoconferenze  
- streaming audio/video  

## Protocolli
La comunicazione su Internet usa protocolli come:
- TCP/IP  
- HTTP/HTTPS per il web  
- SMTP/IMAP/POP3 per la posta elettronica  


# Lezione 8 – Database, query e tabelle pivot

## Cos’è un database
Un **database** è un sistema organizzato per archiviare, gestire e recuperare dati.

Caratteristiche:
- dati strutturati in tabelle (righe = record, colonne = campi)  
- gestione tramite un **DBMS** (Database Management System)  

Esempi di utilizzo:
- anagrafiche clienti  
- magazzino / prodotti  
- fatturazione  
- siti web dinamici

## Database relazionali
Nei database relazionali:
- i dati sono organizzati in tabelle collegate  
- si usano chiavi primarie e chiavi esterne per le relazioni  
- si usa il linguaggio SQL per gestire i dati

## Query
Una **query** è una richiesta al database:
- per leggere dati (selezione, filtri, ordinamenti)  
- per inserire nuovi dati  
- per modificare o cancellare dati  

## Tabelle pivot
Le **tabelle pivot** (soprattutto in Excel o DB + strumenti di analisi) servono per:
- riepilogare grandi quantità di dati  
- raggruppare per categorie (es. per mese, per città, per prodotto)  
- calcolare somme, medie, conteggi  
- creare report e grafici di sintesi  


# Lezione 9 – Sicurezza informatica e competenze digitali per lavoro/studio

## Principi base della sicurezza
Tre concetti fondamentali:
- **Confidenzialità**: i dati sono accessibili solo a chi è autorizzato  
- **Integrità**: i dati non vengono alterati in modo non autorizzato  
- **Disponibilità**: i dati sono disponibili quando servono

## Malware
Il **malware** è software dannoso progettato per:
- danneggiare sistemi  
- rubare o bloccare dati  
- controllare il dispositivo

Tipi principali:
- Virus  
- Worm  
- Trojan  
- Spyware  
- Adware  
- Ransomware  

## Comportamenti sicuri
- usare password robuste e diverse per i servizi importanti  
- non condividere password  
- non aprire allegati o link sospetti  
- aggiornare regolarmente sistema operativo e programmi  
- usare antivirus/antimalware aggiornato  
- eseguire backup periodici dei dati importanti  

## Competenze digitali per studio e lavoro
Competenze essenziali:
- uso consapevole di computer e dispositivi mobili  
- gestione ordinata di file e cartelle  
- utilizzo di Word, Excel e strumenti simili  
- scrittura di email chiare e professionali  
- partecipazione a riunioni online  
- collaborazione su documenti condivisi in cloud  
- consapevolezza dei rischi della rete e delle regole per proteggersi
